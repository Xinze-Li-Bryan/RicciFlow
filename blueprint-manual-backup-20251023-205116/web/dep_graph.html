About this Dependency Graph

What this shows: The logical dependencies between definitions and theorems in the Ricci Flow formalization.

How to read:

  • Each box represents a definition or theorem
  • Arrows show dependencies (A → B means "B depends on A")
  • Green boxes with solid borders are fully formalized in Lean
  • Orange/yellow boxes are definitions that need more work

Nodes in this graph:

  • Manifold Type - Basic manifold structure
  • Riemannian Metric - Metric tensor on manifolds
  • Ricci Tensor - Ricci curvature tensor
  • Scalar Curvature - Trace of Ricci tensor
  • Ricci Flow Equation - The evolution equation
  • Short-Time Existence - Existence theorem for solutions
  • Plus several lemmas: pos_mul_pos, square_pos, inv_pos, etc.
