4 Ricci Curvature

Definition 4.1 Ricci Tensor

blueRicciFlow.RicciTensor The Ricci curvature tensor \(\operatorname{Ric}\) is obtained by contracting the Riemann curvature tensor:

\[ \operatorname{Ric}_{ij} = R^k_{ikj} \]

where \(R^l_{ijk}\) are the components of the Riemann curvature tensor.

Remark 4.2

The current simplified implementation stores only the trace value (scalar curvature). A complete implementation should store all components \(\operatorname{Ric}_{ij}\) of the Ricci tensor.

Definition 4.3 Scalar Curvature

blueRicciFlow.scalarCurvature The scalar curvature \(\operatorname{R}\) is the trace of the Ricci tensor with respect to the metric:

\[ \operatorname{R}= g^{ij} \operatorname{Ric}_{ij} \]

where \(g^{ij}\) denotes the inverse metric tensor.

Geometric interpretation:

  • \(\operatorname{R}{\gt} 0\): The manifold curves inward (like a sphere)

  • \(\operatorname{R}{\lt} 0\): The manifold curves outward (like hyperbolic space)

  • \(\operatorname{R}= 0\): The manifold is locally flat

Lemma 4.4 Scalar curvature extraction

blueRicciFlow.scalarCurvature_eq_traceValue green\(\checkmark \) In the current implementation, the scalar curvature equals the stored trace value.
