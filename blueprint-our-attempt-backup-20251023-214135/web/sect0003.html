2.2 Topological Properties

Lemma 2.5 Continuity equivalence

RicciFlow.continuousAt_iff_continuousWithinAt A function \(f\) is continuous at \(x\) if and only if it is continuous within the universal set at \(x\).

Proof ▼

This follows from continuousWithinAt_univ in Mathlib. Useful when working with smooth functions in local coordinate charts.
