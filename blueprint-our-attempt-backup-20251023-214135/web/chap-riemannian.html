3 Riemannian Manifolds

We introduce the structures of Riemannian geometry.

Definition 3.1 Riemannian Metric

RicciFlow.RiemannianMetric lem:pos_mul_pos, lem:square_pos_of_ne_zero A Riemannian metric on a manifold \(M\) is a smooth assignment to each point \(x \in M\) of a positive-definite symmetric bilinear form \(g_x : T_xM \times T_xM \to \mathbb {R}\).

Formally, it is a structure containing:

  • A function toFun : \(M \to (V \to V \to \mathbb {R})\) where \(V\) represents the tangent space

  • Symmetry: \(g_x(v, w) = g_x(w, v)\) for all \(x, v, w\)

  • Positive-definiteness: \(g_x(v, v) {\gt} 0\) for all \(x\) and all \(v \neq 0\)

Implementation note: The current version uses a type parameter \(V\) for the abstract tangent space. The full implementation should use the actual tangent bundle \(TM\).

Definition 3.2 Inner Product

RicciFlow.innerProduct def:riemannian_metric Given a Riemannian metric \(g\) on \(M\), the inner product of two tangent vectors \(v, w \in T_xM\) at point \(x\) is defined as:

\[ \langle v, w \rangle _x := g_x(v, w) \]
Definition 3.3 Norm Squared

RicciFlow.normSq def:riemannian_metric The squared norm of a tangent vector \(v \in T_xM\) at point \(x\) is:

\[ \| v\| ^2 := g_x(v, v) \]

The actual norm is \(\| v\| = \sqrt{\| v\| ^2}\).

Lemma 3.4 Inner product symmetry

RicciFlow.innerProduct_symm def:inner_product For any tangent vectors \(v, w\) at point \(x\):

\[ \langle v, w \rangle _x = \langle w, v \rangle _x \]
Proof ▼

def:riemannian_metric This follows directly from the symmetry axiom of the Riemannian metric.

Lemma 3.5 Norm squared positivity

RicciFlow.normSq_pos def:norm_sq For any nonzero tangent vector \(v\) at point \(x\):

\[ \| v\| ^2 {\gt} 0 \]
Proof ▼

def:riemannian_metric This follows from the positive-definiteness axiom of the Riemannian metric.
